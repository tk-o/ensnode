Count the Domains owned by an address, grouped by Domain `type` (`ENSv1Domain` vs `ENSv2Domain`) — a single query spanning both protocol versions. See [Connect](/docs/integrate/unigraph/examples) for setup.

<Tabs syncKey="unigraph-client">
  <TabItem label="SQL">
    <EnsDbWriterSchemaIntro ensDbWriterSchemaName="ensindexer_0" />

```sql
SELECT type, count(*) FROM ensindexer_0.domains
WHERE owner_id = '0x70997970c51812dc3a010c7d01b50e0d17dc79c8'
GROUP BY type;
```

    <ResultDetails value={resultJson} />

  </TabItem>

  <TabItem label="ensdb-sdk (TypeScript)">
    <EnsDbReaderIntro />

```typescript
import { count, eq } from "drizzle-orm";

const counts = await ensDb
  .select({ type: ensIndexerSchema.domain.type, count: count() })
  .from(ensIndexerSchema.domain)
  .where(eq(ensIndexerSchema.domain.ownerId, "0x70997970c51812dc3a010c7d01b50e0d17dc79c8"))
  .groupBy(ensIndexerSchema.domain.type);
```
